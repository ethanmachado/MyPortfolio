Reverse Polish Notation Calculator

A Reverse Polish Notation Calculator created using Java. It leverages a Stack data structure to create a calculator that is capable of executing quick calculations. This calculator program does not have to abide by the common paranthesis notation that most calculators need to function properly. What-To-Charger Forecasting Model

An OLS linear regression forecasting model created to forecast the sale price of 1966-1971 Dodge Chargers if they were to be posted on an eBay auction. This model generates a "sale price," but this "sale price" can be used as an approximate value of a car. I created this model to help "Muscle Car Era" Dodge Charger owners put a value to their cars.
